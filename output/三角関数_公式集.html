第4章 三角関数 — 公式集

出典：数学Ⅱ（math2.pdf）

第1節 三角関数

公式
扇形の弧の長さと面積
$$l = r\theta, \qquad S = \frac{1}{2}r^2\theta = \frac{1}{2}lr$$

半径 $r$、中心角 $\theta$（ラジアン）の扇形。弧の長さと面積はともに中心角に比例する。

定義
三角関数の定義
$$\sin\theta = \frac{y}{r}, \qquad \cos\theta = \frac{x}{r}, \qquad \tan\theta = \frac{y}{x}$$

点 P$(x,y)$、原点からの距離 $r$。一般角 $\theta$ に対する正弦・余弦・正接の定義。

公式
三角関数の相互関係
$$① \quad \tan\theta = \frac{\sin\theta}{\cos\theta}$$ $$② \quad \sin^2\theta + \cos^2\theta = 1$$ $$③ \quad 1 + \tan^2\theta = \frac{1}{\cos^2\theta}$$

3つの三角関数を結ぶ基本的な関係式。

公式
三角関数の周期
$\sin k\theta$、$\cos k\theta$ の周期：$\dfrac{2\pi}{k}$   /   $\tan k\theta$ の周期：$\dfrac{\pi}{k}$

$k$ が正の定数のとき。

性質
三角関数の性質 ①（偶奇性）
$$\sin(-\theta) = -\sin\theta, \qquad \cos(-\theta) = \cos\theta, \qquad \tan(-\theta) = -\tan\theta$$

$\sin$・$\tan$ は奇関数（原点対称）、$\cos$ は偶関数（$y$ 軸対称）。

性質
三角関数の性質 ②（$\theta + \pi$）
$$\sin(\theta+\pi) = -\sin\theta, \qquad \cos(\theta+\pi) = -\cos\theta, \qquad \tan(\theta+\pi) = \tan\theta$$

$\pi$ だけずらすと符号が反転（$\tan$ は除く）。

性質
三角関数の性質 ③（$\theta + \dfrac{\pi}{2}$）
$$\sin\!\left(\theta+\frac{\pi}{2}\right) = \cos\theta, \qquad \cos\!\left(\theta+\frac{\pi}{2}\right) = -\sin\theta, \qquad \tan\!\left(\theta+\frac{\pi}{2}\right) = -\frac{1}{\tan\theta}$$

$\dfrac{\pi}{2}$ ずらすと $\sin$ と $\cos$ が入れ替わる。

第2節 加法定理

公式
加法定理（正弦・余弦）
$$\sin(\alpha+\beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta$$ $$\sin(\alpha-\beta) = \sin\alpha\cos\beta - \cos\alpha\sin\beta$$ $$\cos(\alpha+\beta) = \cos\alpha\cos\beta - \sin\alpha\sin\beta$$ $$\cos(\alpha-\beta) = \cos\alpha\cos\beta + \sin\alpha\sin\beta$$

2角の和・差の三角関数を各角の三角関数で表す最重要公式。

公式
加法定理（正接）
$$\tan(\alpha+\beta) = \frac{\tan\alpha + \tan\beta}{1 - \tan\alpha\tan\beta}$$ $$\tan(\alpha-\beta) = \frac{\tan\alpha - \tan\beta}{1 + \tan\alpha\tan\beta}$$

加法定理を $\sin/\cos$ の比に変形して導く。

公式
2倍角の公式
$$\sin 2\alpha = 2\sin\alpha\cos\alpha$$ $$\cos 2\alpha = \cos^2\alpha - \sin^2\alpha = 1 - 2\sin^2\alpha = 2\cos^2\alpha - 1$$ $$\tan 2\alpha = \frac{2\tan\alpha}{1 - \tan^2\alpha}$$

加法定理で $\beta = \alpha$ とおくと得られる。

公式
半角の公式
$$\sin^2\frac{\alpha}{2} = \frac{1 - \cos\alpha}{2}, \qquad \cos^2\frac{\alpha}{2} = \frac{1 + \cos\alpha}{2}, \qquad \tan^2\frac{\alpha}{2} = \frac{1 - \cos\alpha}{1 + \cos\alpha}$$

2倍角の公式 $\cos 2\alpha$ を変形し $\alpha \to \alpha/2$ と置換して導く。

公式
三角関数の合成
$$a\sin\theta + b\cos\theta = \sqrt{a^2+b^2}\,\sin(\theta+\alpha)$$ $$\text{ただし} \quad \cos\alpha = \frac{a}{\sqrt{a^2+b^2}},\quad \sin\alpha = \frac{b}{\sqrt{a^2+b^2}}$$

$\sin$ と $\cos$ の和を1つの $\sin$ にまとめる変形。

発展：和と積の公式

発展公式
積 → 和の公式
$$\sin\alpha\cos\beta = \frac{1}{2}\bigl\{\sin(\alpha+\beta)+\sin(\alpha-\beta)\bigr\}$$ $$\cos\alpha\sin\beta = \frac{1}{2}\bigl\{\sin(\alpha+\beta)-\sin(\alpha-\beta)\bigr\}$$ $$\cos\alpha\cos\beta = \frac{1}{2}\bigl\{\cos(\alpha+\beta)+\cos(\alpha-\beta)\bigr\}$$ $$\sin\alpha\sin\beta = -\frac{1}{2}\bigl\{\cos(\alpha+\beta)-\cos(\alpha-\beta)\bigr\}$$
発展公式
和・差 → 積の公式
$$\sin A + \sin B = 2\sin\frac{A+B}{2}\cos\frac{A-B}{2}$$ $$\sin A - \sin B = 2\cos\frac{A+B}{2}\sin\frac{A-B}{2}$$ $$\cos A + \cos B = 2\cos\frac{A+B}{2}\cos\frac{A-B}{2}$$ $$\cos A - \cos B = -2\sin\frac{A+B}{2}\sin\frac{A-B}{2}$$

加法定理から導かれる積と和の変換公式（発展扱い）。
