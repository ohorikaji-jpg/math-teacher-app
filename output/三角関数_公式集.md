# 第4章 三角関数 — 公式集

出典：数学Ⅱ（math2.pdf）

---

## 第1節 三角関数

### 扇形の弧の長さと面積

**公式名：** 扇形の公式  
**数式：**
$$l = r\theta, \quad S = \frac{1}{2}r^2\theta = \frac{1}{2}lr$$
（半径 $r$、中心角 $\theta$ ラジアン）  
**説明：** 弧の長さと面積はどちらも中心角に比例する。

---

### 三角関数の定義

**公式名：** 三角関数の定義  
**数式：**
$$\sin\theta = \frac{y}{r}, \quad \cos\theta = \frac{x}{r}, \quad \tan\theta = \frac{y}{x}$$
（点 P$(x,y)$、原点からの距離 $r$）  
**説明：** 一般角 $\theta$ に対する正弦・余弦・正接の定義。

---

### 三角関数の相互関係

**公式名：** 三角関数の相互関係  
**数式：**
$$① \quad \tan\theta = \frac{\sin\theta}{\cos\theta}$$
$$② \quad \sin^2\theta + \cos^2\theta = 1$$
$$③ \quad 1 + \tan^2\theta = \frac{1}{\cos^2\theta}$$
**説明：** 3つの三角関数を結ぶ基本的な関係式。

---

### 三角関数の周期

**公式名：** 周期の公式  
**数式：**
- $\sin k\theta$、$\cos k\theta$ の周期：$\dfrac{2\pi}{k}$
- $\tan k\theta$ の周期：$\dfrac{\pi}{k}$

**説明：** $k$ が正の定数のとき、周期はこの式で求まる。

---

### 三角関数の性質（対称・平行移動）

**公式名：** 三角関数の性質 ①（偶奇性）  
**数式：**
$$\sin(-\theta) = -\sin\theta, \quad \cos(-\theta) = \cos\theta, \quad \tan(-\theta) = -\tan\theta$$
**説明：** $\sin$・$\tan$ は奇関数、$\cos$ は偶関数。

---

**公式名：** 三角関数の性質 ②（$\theta + \pi$）  
**数式：**
$$\sin(\theta+\pi) = -\sin\theta, \quad \cos(\theta+\pi) = -\cos\theta, \quad \tan(\theta+\pi) = \tan\theta$$
**説明：** $\pi$ だけずらすと符号が反転（$\tan$ は除く）。

---

**公式名：** 三角関数の性質 ③（$\theta + \dfrac{\pi}{2}$）  
**数式：**
$$\sin\!\left(\theta+\frac{\pi}{2}\right) = \cos\theta, \quad \cos\!\left(\theta+\frac{\pi}{2}\right) = -\sin\theta, \quad \tan\!\left(\theta+\frac{\pi}{2}\right) = -\frac{1}{\tan\theta}$$
**説明：** $\dfrac{\pi}{2}$ ずらすと $\sin$ と $\cos$ が入れ替わる。

## 第2節 加法定理

### 正弦・余弦の加法定理

**公式名：** 加法定理（正弦・余弦）  
**数式：**
$$\sin(\alpha+\beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta$$
$$\sin(\alpha-\beta) = \sin\alpha\cos\beta - \cos\alpha\sin\beta$$
$$\cos(\alpha+\beta) = \cos\alpha\cos\beta - \sin\alpha\sin\beta$$
$$\cos(\alpha-\beta) = \cos\alpha\cos\beta + \sin\alpha\sin\beta$$
**説明：** 2角の和・差の三角関数を各角の三角関数で表す最重要公式。

---

### 正接の加法定理

**公式名：** 加法定理（正接）  
**数式：**
$$\tan(\alpha+\beta) = \frac{\tan\alpha + \tan\beta}{1 - \tan\alpha\tan\beta}$$
$$\tan(\alpha-\beta) = \frac{\tan\alpha - \tan\beta}{1 + \tan\alpha\tan\beta}$$
**説明：** 加法定理を $\sin/\cos$ の比に変形して導く。

---

### 2倍角の公式

**公式名：** 2倍角の公式  
**数式：**
$$\sin 2\alpha = 2\sin\alpha\cos\alpha$$
$$\cos 2\alpha = \cos^2\alpha - \sin^2\alpha = 1 - 2\sin^2\alpha = 2\cos^2\alpha - 1$$
$$\tan 2\alpha = \frac{2\tan\alpha}{1 - \tan^2\alpha}$$
**説明：** 加法定理で $\beta = \alpha$ とおくと得られる。

---

### 半角の公式

**公式名：** 半角の公式  
**数式：**
$$\sin^2\frac{\alpha}{2} = \frac{1 - \cos\alpha}{2}, \quad \cos^2\frac{\alpha}{2} = \frac{1 + \cos\alpha}{2}, \quad \tan^2\frac{\alpha}{2} = \frac{1 - \cos\alpha}{1 + \cos\alpha}$$
**説明：** 2倍角の公式 $\cos 2\alpha$ を変形し $\alpha \to \alpha/2$ と置換して導く。

---

### 三角関数の合成

**公式名：** 三角関数の合成  
**数式：**
$$a\sin\theta + b\cos\theta = \sqrt{a^2+b^2}\,\sin(\theta+\alpha)$$
$$\text{ただし} \quad \cos\alpha = \frac{a}{\sqrt{a^2+b^2}},\quad \sin\alpha = \frac{b}{\sqrt{a^2+b^2}}$$
**説明：** $\sin$ と $\cos$ の和を1つの $\sin$ にまとめる変形。

---

## 発展：和と積の公式

**公式名：** 積→和の公式  
**数式：**
$$\sin\alpha\cos\beta = \frac{1}{2}\{\sin(\alpha+\beta)+\sin(\alpha-\beta)\}$$
$$\cos\alpha\sin\beta = \frac{1}{2}\{\sin(\alpha+\beta)-\sin(\alpha-\beta)\}$$
$$\cos\alpha\cos\beta = \frac{1}{2}\{\cos(\alpha+\beta)+\cos(\alpha-\beta)\}$$
$$\sin\alpha\sin\beta = -\frac{1}{2}\{\cos(\alpha+\beta)-\cos(\alpha-\beta)\}$$

**公式名：** 和・差→積の公式  
**数式：**
$$\sin A + \sin B = 2\sin\frac{A+B}{2}\cos\frac{A-B}{2}$$
$$\sin A - \sin B = 2\cos\frac{A+B}{2}\sin\frac{A-B}{2}$$
$$\cos A + \cos B = 2\cos\frac{A+B}{2}\cos\frac{A-B}{2}$$
$$\cos A - \cos B = -2\sin\frac{A+B}{2}\sin\frac{A-B}{2}$$
**説明：** 加法定理から導かれる積と和の変換公式（発展扱い）。
